How CSS Works

CSS is also a language and has its own specific syntax and rules. CSS is used to change how elements look on the page. By adding styles with CSS you can change font, color and any number of other attributes.

All About Boxes

Every Element in a webpage is a box. Having elements within elements creates boxes within boxes These boxes make it easier to arrange or re-arrange your page. Div elements and P elements are types of Block elements that create the boxes.
You can use Class Attributes to label your boxes which will make it easier to identify them when you are ready to begin adding styles.

Lesson 3 - Adding CSS Style to HTML Structure
Don't repeat yourself

Don't repeat yourself. If you use the same code over and over you are more likely to make mistakes and those mistakes will be harder to find. MIX IT UP!!!!!

What is CSS?

CSS is code written to control the "style" of HTML elements.

How does CSS "control" HTML?

CSS controls HTML through the use of selectors. Selectors determine if styling is added to the entire element, Specific Tags or class Arttributes

How do I include CSS Styling in my web page?

There are 3 ways to get CSS styling in your document:
Method 1: write CSS in the head of your HTML. This is best for small projects
Method 2: Link your HTML to a separate CSS file. This adds a step but allows better organization (requires HTML and CSS files to be in the same folder for this to work)
Method 3: Write your style inline with your HTML. Not suggested this is a very cumbersome process and requires a lot of repetitive code (something we avoid)
